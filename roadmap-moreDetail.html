Front-End Web Developer ជាអ្នកដែលអភិវឌ្ឍផ្នែកខាងមុខនៃ website។ ពួកគេមានតួនាទី រៀបចំ structure និង ការរចនារបស់ website application, build features សម្រាប់ឱ្យមានភាពប្រសើរឡើងទៅឱ្យអ្នកប្រើប្រាស់។ លើសពីនេះទៅទៀតពួកគេទទួលខុសត្រូវក្នុងការធ្វើឱ្យ website មានភាពស្រស់ស្អាតនិង មានសភាពល្អ (មិនខូចទម្រង់) នៅពេលដែល website បើកនៅលើ devices ផ្សេងៗគ្នា។

ពួកគេមិនមែនកំណត់ឱ្យចេះត្រឹម HTML, CSS ឬ JS នោះទេ គឺតម្រូវឱ្យមានចំណេះដឹងទៅលើ technology មួយចំនួនផងដែរ​ (យើងនឹងបង្ហាញនៅចំណុចក្រោយ)។

  • CSS (Cascading Style Sheets)

  • CSS Preprocessors (Optional but recommended)
